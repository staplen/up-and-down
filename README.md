# Up & Down the River

A mobile-first scoring companion web app for the card game "Up and Down the River."

## What it does

Tracks bids, tricks, and scores across all 14 rounds while you play the physical card game. Designed for the scorekeeper to manage on their phone at the table.

## How to use

Open `index.html` in a browser, or upload the files to any static web host.

No build step, no dependencies, no backend. Just HTML, CSS, and vanilla JS.

## Game rules

- 2-7 players, standard 52-card deck
- Rounds go 1-2-3-4-5-6-7-7-6-5-4-3-2-1 cards (14 rounds total)
- First half is "Up the River," second half is "Down the River"
- Scoring: make your bid = (bid × 5) + 10 points; miss = 0 points
